Sample of Early Mapping Work

Breweries in North America

This is a hexbin map that shows the density of breweries across North America.

The map was created with QGIS using the North American Albers Equal Conic (EPSG:102008) CRS.

  • See the map: Breweries in North America
  • Data Source: POI Factory

Cancer Mortality Rates

This is choropleth map depicting the rate of all cancer deaths, by county, across the United States, averaged over the years 2008 - 2012.

The map was created using CartoDB and using the Albers Equal Area Conic projection. Users may click on an area of the map to see a the county name, the name of the state, and the number of cancer deaths.

  • See the map: Cancer Mortality Rates
  • Data Source: National Cancer Institute's State Cancer Profiles

Population of Selected Florida Cities

This is a very simple map showing the location and population of three cities in Florida.

The map was created using JavaScript, with the base map provided the Leaflet JavaScript Library. When users click on the popup, the name of the city and its population is shown.

  • See the map: Population of Selected Florida Cities
  • Data Source: 2010 U.S. Census
